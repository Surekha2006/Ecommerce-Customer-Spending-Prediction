# Ecommerce-Customer-Spending-Prediction

This project uses Linear Regression to predict a customer's yearly spending based on their online activity.

Tools Used:
Python
Pandas
Seaborn
Scikit-Learn
Jupyter Notebook

Features:
Avg. Session Length
Time on App
Time on Website
Length of Membership

Target:
Yearly Amount Spent

Result:
The model achieved an R² score of approximately 0.98 and identified Length of Membership as the most influential feature.
